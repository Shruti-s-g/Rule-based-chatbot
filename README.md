# Rule-based-chatbot


This is a simple Python-based rule-driven chatbot created to practice programming
logic and user interaction.

The chatbot responds to user input using predefined rules and dictionary-based
matching. It also provides time-based greetings using the datetime module.

## Features
- Rule-based responses using dictionaries
- Time-based greeting (morning/afternoon/evening/night)
- Continuous conversation until user exits

## Technologies Used
- Python

## Concepts Covered
- Conditional statements
- Dictionaries
- Functions
- Loops
- User input handling

## Learning Source
This project was built while learning Python concepts from online tutorials.
The code was written and understood step by step for educational purposes.
